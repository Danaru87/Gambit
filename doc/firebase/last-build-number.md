# last-build-number

Get last build number from Testflight.

```bash
gambit help firebase last-build-number
```
 
 ```bash
Get last build number from App Distribution releases.

Usage: gambit firebase last-build-number [arguments]
-h, --help                                Print this usage information.
-f, --account-service-file (mandatory)    Path to your .json service account file.
-p, --project-id (mandatory)              Firebase project id.
-a, --app-id (mandatory)                  Firebase app id.
-n, --default-number                      If no release found, will return this value.

Run "gambit help" to see global options.
```
