# get-app-version

```
gambit get-app-version -h
Read flutter app semantic version from pubspec.

Usage: gambit get-app-version [arguments]
-h, --help       Print this usage information.
-f, --folder     Path to flutter app root folder,
                   if not set will try to find flutter pubspec.yaml in current path.
                 (defaults to "$pwd")
-v, --verbose    Verbose mode, use only for debug
```
